# 图片资源说明

本目录存放小程序所需的图片资源。

## 目录结构

```
images/
├── icon/           # 图标资源
│   ├── habit.png   # 习惯图标
│   ├── note.png    # 笔记图标
│   ├── stats.png   # 统计图标
│   └── calendar.png # 日历图标
├── default-avatar.png  # 默认头像
└── placeholder.png    # 占位图
```

## 图片规格

- 图标：建议 64x64 或 128x128 px
- 头像：建议 200x200 px
- 占位图：建议 800x600 px

## 注意事项

1. 请使用PNG或JPG格式
2. 图片大小建议控制在 200KB 以内
3. 图标建议使用透明背景
4. 建议使用微信小程序官方推荐的图片尺寸

## 临时方案

在正式图片资源准备完成前，可以使用以下方案：

1. 使用emoji代替图标
2. 使用网络图片URL
3. 使用纯色背景+文字
